# Standalone Mode

In standalone mode, the Modalix DevKit runs as a self-contained edge device. It receives data from sensors, cameras, storage, or network interfaces, processes it locally, and sends results over the network.

Use this section when the DevKit is not installed in a host machine as a PCIe card.
