# PCIe Mode

In PCIe mode, the Modalix PCIe Card is installed in a host machine and provides Modalix inference acceleration over PCIe. The host handles orchestration, storage, and I/O, while the card handles inference workloads.

Use this section when the Modalix hardware is installed as a PCIe card rather than running as a standalone DevKit.
