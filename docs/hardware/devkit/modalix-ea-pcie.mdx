# Modalix Early Access PCIe Card

The Modalix PCIe HHHL (Half-Height/Half-Length) Early Access board integrates the Modalix device that delivers high-performance and effortless machine learning inference for embedded edge applications.

<Tabs>
<TabItem value="interfaces" label="Interfaces" default>

<img src="/img/devkit/modalix-ea-pcie-interfaces.jpg" alt="SiMa.ai Modalix EA PCIe Card Interfaces" style={{width: '100%', display: 'block', margin: '0 auto'}} />

| Interface | Description |
| --- | --- |
| Ethernet Ports (eth0, eth1) | RJ45, GigE network interfaces for remote connectivity. |
| PCI-Express 5.0 x4 Interface | High-speed PCIe interface used for integration into a host system. Adapters may be required depending on the host configuration. |
| COM Port to Host | Facilitates debugging by connecting to the host system. Ensure the arrow sign on the included cable connects to the left-most pin for proper alignment. |
| SD Card Slot | The SD Card slot is located at the back of the Modalix EA PCIe card. |
| GMSL2 FAKRA Port for Camera | Gigabit Multimedia Serial Link ensures stable high-bandwidth video transmission. |

</TabItem>
<TabItem value="features" label="Key Features">

| Feature | Details |
| --- | --- |
| PCIe Interface | One PCIe Gen 5.0 x4 link interface with a standard board-edge connector. |
| System Memory | 16GB or 32GB LPDDR5 DRAM interface for high-bandwidth, low-latency operation. |
| eMMC Flash Storage | One 32GB eMMC flash memory interface for boot and system storage. |
| QSPI NOR Flash | One 128Mbit QSPI NOR flash interface for firmware and bootloader storage. |
| Ethernet | One 10Gbps Ethernet interface via RJ45 connector. |
| JTAG Interface | One optional JTAG interface for hardware debug and bring-up. |
| Micro SD Interface | One Micro SD card interface for removable storage and system recovery. |
| Debug UART (Troot) | One dedicated debug UART interface for Troot diagnostics and debugging. |
| Debug UART (Linux) | One dedicated debug UART interface for Linux console access and debugging. |
| Camera Interfaces | Two GMSL2 FAKRA ports for connecting remote cameras over serialized links. |
| Status Indicator | One power LED for system power status indication. |

</TabItem>
