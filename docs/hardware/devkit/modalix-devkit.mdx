# Modalix DevKit

The **Modalix DevKit** is a compact, high-performance development platform designed for rapid prototyping and evaluation of embedded edge AI applications. Built on the powerful Modalix platform, the DevKit combines advanced hardware acceleration with an optimized software stack to enable efficient, flexible, and scalable AI inference across workloads such as computer vision, large language models (LLMs), and real-time multimodal processing.

<TabItem value="features" label="Key Features">

| Feature | Details |
| --- | --- |
| MLA AI Performance | 50 Tera Operations per Second for INT8 neural network computation. Also supports bf16 precision. |
| Power Consumption | 8W to 12W depending on workload |
| Application Processing Unit (APU) | A cluster of eight [Arm Cortex-A65 CPU](https://www.arm.com/products/silicon-ip-cpu/cortex-a/cortex-a65) dual threaded processors operating up to 1.4 GHz to deliver up to 32K Dhrystone MIPs Eight-core at 1.4GHz. |
| Video Encoder | Supports the H.264 (AVC), H.265 (HEVC) and MJPEG compression standards. It has support for AVC main/high profiles, HEVC main profile, and MJPEG baseline profile. The encoder supports H.264/H.265 at rates up to 4KP60. |
| Video Decoder | supports the H.264 (AVC), H.265 (HEVC), AV1, and MJPEG compression standards. It has support for AVC main/high profiles, HEVC main profile, AV1 main profile, and MJPEG Baseline profile. It supports 4:2:0 pixels and 8-bit precision. The decoder supports H.264/H.265 at rates up to 4KP60. |
| Computer Vision Unit (CVU) | Four-core Synopsys [ARC EV74 Vision Processor](https://www.synopsys.com/dw/ipdir.php?ds=ev7x-vision-processors) supporting up to **750 16-bit GOPS**. |
| Memory | 32GB LPDDR5 RAM, of which 6GB allocated to the ARM cores and 16GB allocated to the MLA. Rest is allocated to other parts of the system. |
| Storage | 16GB internal eMMC with microSD card slot for additional storage |
| Networking | 1X 10GigE ports available through carrier board |
