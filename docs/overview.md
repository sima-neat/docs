# SiMa.ai System Documentation

This repository is the home for system-level hardware documentation for SiMa.ai platforms.

The first content areas are focused on:

- Hardware platform overview
- Modalix DevKit setup
- Serial, network, storage, and firmware bring-up notes
- Field-ready troubleshooting references

The structure is intentionally small for now so the Vulcan sysdoc deployment path can be validated before the content set grows.
