# Hardware Reference Docs Portal

This page is the entry point for detailed hardware reference material as the system documentation portal grows.

Reference material should include board-specific information that is useful during integration and debug:

- Connector and interface references
- Board boot and recovery notes
- Power, storage, and networking details
- Serial console behavior
- Firmware and image compatibility notes
- Known hardware-specific troubleshooting steps

Until the full reference portal is populated, use the hardware overview and DevKit setup pages as the primary bring-up path.
