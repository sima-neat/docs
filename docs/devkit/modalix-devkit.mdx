# Modalix DevKit

The **Modalix DevKit** is the primary development kit for evaluating MLSoC Modalix and building edge AI applications. It combines Modalix hardware acceleration with the software stack needed to develop, test, and run workloads such as computer vision, large language models (LLMs), and multimodal pipelines.

<Tabs>
<TabItem value="interfaces" label="Interfaces" default>

<img src="/img/devkit/modalix-devkit.png" alt="SiMa.ai Modalix SoM Interfaces" style={{width: '60%', display: 'block', margin: '0 auto'}} />

| Interface | Description |
| --- | --- |
| Ethernet Port (end0) | RJ45, 1GigE network interfaces for remote connectivity. |
| COM Port to Host (USB-C) | Connects the serial console to the host for debugging. |
| MIPI Interfaces (X2) | Connect MIPI cameras. See [MIPI Camera Interfaces](/hardware/getting-started/standalone-mode/mipi-camera-interfaces). |
| USB Interfaces (X4) | USB 3.0 interfaces for microphones, speakers, and USB video cameras. |
| HDMI Interface (X1) | HDMI 1.4 display output (max 4K at 30 Hz, 1080p at 60 Hz). |
| Onboard storage (eMMC) | 16GB |
| NVMe | M.2 512GB |
| USB | 4x USB 3.0 ports |

For more product information, visit the official website: [SiMa.ai](https://sima.ai/mlsoc/).

</TabItem>
<TabItem value="features" label="Key Features">

| Feature | Details |
| --- | --- |
| MLA AI Performance | 50 Tera Operations per Second for INT8 neural network computation. Also supports bf16 precision. |
| Power Consumption | 8W to 12W depending on workload |
| Application Processing Unit (APU) | Eight dual-threaded [Arm Cortex-A65 CPU](https://www.arm.com/products/silicon-ip-cpu/cortex-a/cortex-a65) cores operating up to 1.4 GHz. |
| Video Encoder | Supports H.264 (AVC), H.265 (HEVC), and MJPEG compression. Supports AVC main/high profiles, HEVC main profile, MJPEG baseline profile, and H.264/H.265 rates up to 4KP60. |
| Video Decoder | Supports H.264 (AVC), H.265 (HEVC), AV1, and MJPEG compression. Supports AVC main/high profiles, HEVC main profile, AV1 main profile, MJPEG baseline profile, 4:2:0 pixels, 8-bit precision, and H.264/H.265 rates up to 4KP60. |
| Computer Vision Unit (CVU) | Four-core Synopsys [ARC EV74 Vision Processor](https://www.synopsys.com/dw/ipdir.php?ds=ev7x-vision-processors) supporting up to **750 16-bit GOPS**. |
| Memory | 32GB LPDDR5 RAM, with 6GB allocated to the Arm cores and 16GB allocated to the MLA. The remaining memory is allocated to other system components. |
| Storage | 16GB internal eMMC with microSD card slot for additional storage |
| Networking | 1X 10GigE ports available through carrier board |

</TabItem>
