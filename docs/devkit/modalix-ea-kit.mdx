# Modalix Early Access DevKit

The **Modalix Early Access DevKit** is a legacy development kit for existing Early Access customers. It supports evaluation and prototyping on Modalix hardware for workloads such as computer vision, large language models (LLMs), and real-time multimodal processing.

<Tabs>
<TabItem value="interfaces" label="Interfaces" default>

<img src="/img/devkit/modalix-ea-kit-interfaces.png" alt="SiMa.ai Modalix Early Access Kit Interfaces" style={{width: '80%', display: 'block', margin: '0 auto'}} />

| Interface | Description |
| --- | --- |
| Ethernet Ports (eth0, eth1, eth2, eth3) | RJ45, 10GigE network interfaces for remote connectivity. |
| 12V Power (J8) | Connect the included 12V power adapter to this port. |
| Power On/Off Switch | Located next to the power connector. Use a pen to reach the switch if needed. |
| COM Port to Host | Connects the serial console to the host for debugging. Align the arrow on the included cable with the left-most pin. |
| Reset Button | Available in standalone mode only. Resets the device for a fresh boot or recovery. |
| MIPI Interfaces (X4) | Connect cameras. |
| Micro SD Card Slot | Provides additional storage. |

For more product information, visit the official website: [SiMa.ai](https://sima.ai/mlsoc/).

<TabItem value="features" label="Key Features">

| Feature | Details |
| --- | --- |
| MLA AI Performance | 50 Tera Operations per Second for INT8 neural network computation. Also supports bf16 precision. |
| Power Consumption | 8W to 12W depending on workload |
| Application Processing Unit (APU) | Eight dual-threaded [Arm Cortex-A65 CPU](https://www.arm.com/products/silicon-ip-cpu/cortex-a/cortex-a65) cores operating up to 1.4 GHz. |
| Video Encoder | Supports H.264 (AVC), H.265 (HEVC), and MJPEG compression. Supports AVC main/high profiles, HEVC main profile, MJPEG baseline profile, and H.264/H.265 rates up to 4KP60. |
| Video Decoder | Supports H.264 (AVC), H.265 (HEVC), AV1, and MJPEG compression. Supports AVC main/high profiles, HEVC main profile, AV1 main profile, MJPEG baseline profile, 4:2:0 pixels, 8-bit precision, and H.264/H.265 rates up to 4KP60. |
| Computer Vision Unit (CVU) | Four-core Synopsys [ARC EV74 Vision Processor](https://www.synopsys.com/dw/ipdir.php?ds=ev7x-vision-processors) supporting up to **750 16-bit GOPS**. |
| Memory | 64GB LPDDR5 RAM, with 10GB allocated to the Arm cores and 40GB allocated to the MLA. The remaining memory is allocated to other system components. |
| Storage | 10GB internal eMMC with microSD card slot for additional storage |
| Networking | 4X 10GigE ports. |
